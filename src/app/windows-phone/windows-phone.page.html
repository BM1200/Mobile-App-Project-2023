Windows Phone (Brief Overview)

The {{phone}} was a smartphone operating system developed by Microsoft, first released in 2010. It was designed to compete with iOS and Android but struggled to gain traction in the market. Despite several updates and attempts to gain popularity, Microsoft eventually announced the discontinuation of the Windows Phone in 2017.
